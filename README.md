# 🧺 LavExpress - Lavandería a Domicilio

Demo funcional de app móvil para servicio de lavandería a domicilio en Veracruz.

## 📱 Características

✅ Interfaz móvil responsiva  
✅ Selección de servicios (Lavado, Secado, Planchado, Doblado)  
✅ Programación de recolección y entrega  
✅ Seguimiento en tiempo real de pedidos  
✅ Historial de pedidos  
✅ Gestión de perfil de usuario  
✅ Diseño moderno con animaciones  

## 🚀 Cómo Usar

1. Abre `index.html` en el navegador
2. Explora las pantallas:
   - 🏠 **Inicio**: Resumen de servicios y pedido activo
   - ➕ **Nuevo Pedido**: Crear un nuevo pedido paso a paso
   - 📍 **Estado**: Ver estado actual de tu pedido
   - 📋 **Historial**: Tus pedidos anteriores
   - 👤 **Perfil**: Información personal

## 🛠️ Stack Técnico

- **Frontend**: HTML5, CSS3, JavaScript puro (Vanilla JS)
- **Base de Datos**: (Simulada en v1 - integración en próxima versión)

1. **Home** - Vista principal con servicios activos

- Lavado: desde $35/kg
- Secado: desde $25/kg  
- Planchado: desde $15/pieza
- Doblado: desde $10/kg
